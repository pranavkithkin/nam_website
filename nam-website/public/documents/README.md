# NAM UAE Documents

This directory contains all company documents, profiles, catalogs, and brochures organized professionally.

## Directory Structure

### `/profiles` - Company Profiles
Official company profiles and capability statements
- `NAM-Corporate-Profile.pdf` - Complete corporate profile and company overview
- `NAM-Promotional-Gifts-Profile.pdf` - Promotional gifts and corporate gifting solutions profile

### `/catalogs` - Product Catalogs
Comprehensive product catalogs showcasing offerings
- `NAM-Product-Catalog.pdf` - Complete product catalog with all services and products

### `/brochures` - Service Brochures
Detailed brochures for specific services
- `NAM-Printing-Services.pdf` - Digital printing and printing services brochure
- `NAM-Uniform-Printing.pdf` - Custom uniform printing and branding brochure

## Usage in Website

These documents can be accessed via URLs:
- Corporate Profile: `/documents/profiles/NAM-Corporate-Profile.pdf`
- Promotional Gifts Profile: `/documents/profiles/NAM-Promotional-Gifts-Profile.pdf`
- Product Catalog: `/documents/catalogs/NAM-Product-Catalog.pdf`
- Printing Services: `/documents/brochures/NAM-Printing-Services.pdf`
- Uniform Printing: `/documents/brochures/NAM-Uniform-Printing.pdf`

## Recommended Links to Add

Consider adding download links on these pages:
- **About page**: Link to Corporate Profile
- **Services pages**: Link to relevant service brochures
- **Contact page**: Link to Product Catalog
- **Footer**: Link to Corporate Profile or Product Catalog

## File Naming Convention

Format: `NAM-[Document-Type].pdf`
- Use hyphens for multi-word names
- Keep names clear and descriptive
- Include "NAM" prefix for brand consistency
